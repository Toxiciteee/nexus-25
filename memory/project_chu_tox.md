Web application for the **Service de Toxicologie, CHU Constantine**.

**Stack:** Next.js 15 (App Router) + TypeScript + Tailwind v4 + shadcn/ui + Supabase (PostgreSQL + Auth + RLS + Storage) + lucide-react + react-hook-form + zod + @react-pdf/renderer.

**Working directory:** `/Users/lindabenboudiaf/nexus-25`

**Key people:**
- Mme **Benboudiaf Sabah** — Chef de Service (top-level validator, only one with admin access).

**5 Units (`unites`):**
1. Urgences toxicologiques
2. Médico-légale
3. Pharmacodépendance
4. Suivi thérapeutique
5. Contrôle cosmétiques

**Why:** Replace paper/manual workflow with a digital chain of custody from sample registration to signed PDF report.

**How to apply:** When in doubt about scope or naming, prefer French (UI is French-only for now; Arabic to be added later for the printed final report only).
